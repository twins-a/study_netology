1. Загрузка файла:

wget https://files.grouplens.org/datasets/movielens/ml-latest-small.zip

2. Установил unzip:
sudo apt install unzip

3. Разархивировал:
unzip ml-latest-small.zip

4. Посмотрел имя контейнера:
sudo docker ps

5. Скопировал файл ratings.csv в контейнер:
sudo docker cp ratings.csv festive_diffie:/ratings.csv



6. Посчитал топ-5 пользователей, которые выставили наибольшее количество оценок:
sudo cut -d ',' -f1 ratings.csv | uniq --count | sort -nrk1 | head -n 5

а можно так:
sudo cut -d ',' -f1 ratings.csv | uniq --count | sort -nk1 | tail -n 5


Максимальное количество оценок в количестве 2698 выставил пользователь с id 414
