cd kafka\bin\windows


1. Запускаю zookeeper из каталога c:\Kafka\bin\windows\
	zookeeper-server-start.bat ../../config/zookeeper.properties

2. запускаю apache Kafka
	kafka-server-start.bat ../../config/server.properties
	
3. Создаю Kafka topic (один раз)
	for win10: kafka-topics -–create –-topic net-topic -–bootstrap-server localhost:9092
	for win7 :  kafka-topics --create --zookeeper localhost:2181 --replication-factor 1 --partitions 1 --topic net-topic
	
4. Инициализация producer
	kafka-console-producer --topic net-topic --bootstrap-server localhost:9092

5. Инициализация consumer
	kafka-console-consumer --topic net-topic --from-beginning --bootstrap-server localhost:9092

6. Устанавливаю Spark и проверяю
	spark-shell

7. Установил и настроил pycharm

8. Установил kafka-python + pyspark
	pip install kafka-python
	pip install pyspark
	
9. Небольшие изменения в файле producer.py и запускаю его
	python producer.py
	python кидает сообщения, а consumer принимает их

10. В cmd запускаю 
	spark-submit --packages org.apache.spark:spark-sql-kafka-0-10_2.12:3.1.2 c:\Users\User_TA\PycharmProjects\lesson2\streaming.py

	в pycharm запускаю producer.py

В итоге в cmd ловлю сообщения (см скрин)
